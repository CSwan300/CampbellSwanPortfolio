## ğŸš€ Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:
- **Node.js** (v18.0.0 or higher)
- **npm** (v9.0.0 or higher) or **yarn**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/CSwan300/CampbellSwanPortfolio.git
   cd CampbellSwanPortfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` to view the portfolio

## ğŸŒ Deployment

This portfolio is deployed on **Vercel** at [campbellswan.vercel.app](https://campbellswan.vercel.app/)

### Deploy Your Own

#### Vercel (Recommended)
1. Push your code to GitHub
2. Import your repository on [Vercel](https://vercel.com)
3. Vercel will automatically detect Vite and configure the build
4. Your site will be live with automatic deployments on every push

#### GitHub Pages
1. **Build the project**
   ```bash
   npm run build
   ```

2. **Deploy to GitHub Pages**
   ```bash
   npm install -g gh-pages
   gh-pages -d dist
   ```

#### Other Platforms
The built application in `dist/` can be deployed to:
- **Netlify** - Drag and drop the `dist/` folder
- **AWS S3** - Upload to an S3 bucket with static hosting
- **Firebase Hosting** - Use Firebase CLI to deploy
